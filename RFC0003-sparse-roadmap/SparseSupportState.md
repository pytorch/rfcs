# Flags

Flags are defined in the FLAGS section of <a
href="pytorch_functions.ini">pytorch_functions.ini</a>.


| Flag |                          Description                           | Usage count |
| :--- | :------------------------------------------------------------- | :---------- |
| D    | dense-only function, supports only strided layout              | 2           |
| S    | sparse-only function, supports only sparse layout              | -           |
| !    | support for a sparse layout is required                        | 33          |
| F    | support for a sparse layout is required, blocked by fill-value | 24          |
| ?    | support for a sparse layout is questionable/irrelevant         | 20          |
| I    | not applicable, ignore                                         | 52          |
|      | no flag specified                                              | 500         |
